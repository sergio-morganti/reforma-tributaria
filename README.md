# Reforma Tributária (IBS/CBS/IS) — Plugin Claude/Cowork

Suite de skills para operacionalizar a Reforma Tributária do Consumo brasileira (EC 132/2023, LC 214/2025, LC 227/2026).

## Skills

| Skill | O que faz |
|---|---|
| **monitor-reforma-tributaria** | Varre fontes oficiais e secundárias, detecta novidades e gera resumo executivo em PDF. |
| **valida-nfe** | Recebe XML de NF-e/NFC-e, recalcula os tributos (ICMS, IPI, PIS, COFINS, IS, IBS/CBS) e identifica retenções aplicáveis. |
| **impacto-sped** | Recebe arquivos SPED e estima carga atual vs. pós-reforma (2026-2033), créditos e impacto em preço e margem. |
